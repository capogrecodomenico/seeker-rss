# Seeker RSS App Description

## Short Description

Seeker RSS is a privacy-conscious Android RSS reader for subscribing to feeds, importing OPML, organizing articles, and optionally syncing with compatible RSS services.

## Full Description

Seeker RSS is an Android RSS reader designed to help users discover, organize, and read web content from RSS feeds in one place.

Core features include:
- subscribe to RSS feeds using a feed URL or website URL;
- search for RSS sources by keyword and add matching feeds;
- import and export subscriptions using OPML;
- manage multiple accounts, including local-only use and optional compatible third-party or self-hosted RSS services;
- organize feeds into groups and track unread or starred articles;
- enable optional notifications for new unread articles;
- improve article readability and, where available, parse fuller article content for a better reading experience;
- customize reading and appearance settings for a cleaner mobile reading workflow.

Seeker RSS is a content reader and organizer. It is not:
- a wallet, custodian, exchange, or broker;
- a financial or investment product;
- a gambling service;
- a medical or healthcare application; or
- a social platform with built-in public user-generated posting or review functionality.

### Data and privacy summary

Seeker RSS primarily stores reading data, preferences, and optional account settings locally on the user's device. If a user chooses to connect third-party or self-hosted RSS services, or uses RSS source search, the App will send the necessary requests to those services to fulfill the user's request.

The App does not claim affiliation with any feed publisher, third-party synced service, or Solana Mobile unless expressly stated.

### Intended audience

Seeker RSS is intended for users who want a customizable RSS reading experience on Android, including users who prefer local-only reading as well as users who want to connect supported RSS backends.
